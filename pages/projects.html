3D Voxel Completion Transformer
2024 – Present

A transformer architecture designed to perform 3D object completion by predicting missing voxels from partial volumetric inputs. The model handles the unique challenges of sparse 3D data through custom attention mechanisms and CUDA-level optimizations.

  • Trained on 5M examples and validated on 1M examples with strong generalization — training loss 0.02, validation loss 0.01.
  • Achieved 99% accuracy on both filled and empty voxel classification.
  • Conducted ablation studies comparing learned positional embeddings vs. sinusoidal encodings across varying mask rates to characterize semantic representation quality.
  • Implemented custom spatial attention mechanisms using CUDA-accelerated kernels to handle sparse volumetric data efficiently.

Lebanese Sign Language Translator
Aug 2023 – Jun 2024

An end-to-end sign language recognition system for Lebanese Sign Language (LSL) — a previously undocumented domain with no existing datasets. The project covers dataset creation, model training, a mobile app, and a public website.

  • Pioneered the first dataset for Lebanese Sign Language, enabling machine learning research in a previously undocumented domain.
  • Used MediaPipe for landmark extraction and OpenCV for preprocessing; trained XGBoost classifiers on extracted hand keypoints.
  • Achieved 97% accuracy for static symbols and 100% accuracy for dynamic signs.
  • Shipped a cross-platform mobile app (Flutter) and a public-facing website for broader accessibility.
